Pinbelegung:

Motor1 = Hinten Links
Motor2 = Vorne Links
Motor3 = Vorne Rechts
Motor4 = Hinten Rechts
Farbsensor 1 = Hinten
Farbsensor 2 = Vorne
Farbsensor 3 = Rechts
Farbsensor 4 = Links

Befehle:

vorne() = Nach vorne fahren
stop() = Aus
drehenl() = Links drehen
drehenr() = Rechts drehen
readUs([Sensor]) = Ultraschall abfragen
selectChannel([1-4]) = Farbsensor auswählen
red([1-4]) = Farbsensor Rot abfragen
green([1-4]) = Farbsensor grün abfragen
blue([1-4]) = Farbsensor blau abfragen
